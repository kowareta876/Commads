Commands

Music Commands

!songs - View the various commands for music

!q - View the song queue

!next - See what song will play next

!current - See what song is currently playing

Message Commands

!matchmaker - Play matchmaker for stream viewers

!hug - Hug a specific user in chat

!rhug - Hug a random user in chat

!lewd - Lewd a specific user in chat

!rlewd - Lewd a random user in chat

Curses Commands

!points - View a user’s amount of Curses and rank

!duel - Duel another player for Curses

!givepoints - Donate Curses to another viewer

!items - View the redeemable stream items

!redeem - Redeem a stream item

!rename - Transfer your points if you’ve changed your username

!roulette - Play the roulette

!slots - Play the slots

!top - Show the top of the leaderboard

Info Commands

!chatstats - View the stream’s chat statistics

!followage See how long a viewer has been following

!uptime - View the uptime of the stream

!watchtime - Check how long a user has been watching the stream

!accountage - See when a user’s account was created

!tip - Make a donation

Other Commands

!8ball - Ask the magic 8-ball a question

!randomnumber - Generate a random number

!quote - View a random quote or enter a number afterwards to view a specific quote

!addquote - Add a quote from chat or stream, don’t forget to write you said it
